# 寻忆 - 前端设计文档

## 项目概述

**名称：** 寻忆 (Memory Seek)
**类型：** 照片展示、管理网站
**功能：** 浏览、收藏、评论、点赞

## 设计规范

### 配色系统

| 用途 | 浅色模式 | 深色模式 |
|------|----------|----------|
| **背景** | `#FFFFF0` 象牙白 | `#121212` 深黑 |
| **表面** | `#FFFFFF` 纯白 | `#1E1E1E` 深灰 |
| **主色** | `#2DD4A8` 青绿 | `#2DD4A8` 青绿 |
| **辅助色** | `#818CF8` 淡紫 | `#818CF8` 淡紫 |
| **文字主** | `#1F2937` 深灰 | `#F9FAFB` 浅灰 |
| **文字次** | `#6B7280` 中灰 | `#9CA3AF` 中灰 |
| **边框** | `#E5E7EB` 浅灰 | `#374151` 深灰 |

### 字体

- **主字体：** 思源黑体 (Noto Sans SC)
- **备用字体：** -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif

### 设计风格

- **极简主义：** 大量留白、简洁线条、专注内容
- **圆角：** 8px
- **阴影：** 轻柔投影

## 页面结构

### 1. 首页

- 顶部导航栏（桌面端）
- 瀑布流照片展示
- 无限滚动加载

### 2. 个人主页

- 用户头像 + 昵称 + 简介
- 统计数据：点赞数、收藏数、照片数
- Tab 切换（默认选中"我喜欢"）：
  - 我喜欢
  - 收藏夹
  - 我的照片

### 3. 照片详情页

- 大图展示
- 操作按钮：点赞、收藏、分享
- 评论区（默认折叠，点击展开）

## 组件设计

### 导航栏

**桌面端：**
- 固定顶部
- Logo（青绿色）+ 导航链接 + 搜索 + 头像
- 当前页面高亮

**移动端：**
- 底部导航栏
- 4个图标：首页、喜欢、收藏、我的

### 照片卡片

**桌面端：**
- 悬停显示遮罩层 + 操作按钮（点赞、收藏、分享）
- 底部显示：点赞数、评论数

**移动端：**
- 操作按钮常驻卡片底部
- 简化显示

## 响应式设计

- **桌面端：** ≥ 1024px
- **平板端：** 768px - 1023px
- **移动端：** < 768px

## 交互设计

- 照片卡片点击进入详情页
- 评论区点击展开/收起
- 无限滚动加载更多照片
- 主题切换（浅色/深色）

## 技术栈

- **框架：** Vue 3
- **样式：** Tailwind CSS
- **字体：** 思源黑体 (Google Fonts)
